The additional features we added:

Dark Mode:A feature that allows the page to switch between dark and light modes by clicking on moon/sun icons.

Dog of the day: A feature where a picture is randomly selected from all the photos uploaded by users during registration and is displayed on the Feed Page. The picture changes every 24 hours.

What was hard to do: The entire project was a significant challenge as I had never worked with Node.js, Express, and JavaScript before. In addition, never dealt with building a whole project from scratch. The most challenging parts for me were at first to structure the project, then managing the routes and handling data on the local disk with JSON files. Afterwards, building the Feed Page and fetching all user's posts and assigning for each post his username and displaying it on screen. Additionally, managing the "Following Users" Page and the Admin Page posed challenges, but after overcoming all the obstacles and errors, I was really proud of myself and I really liked that in web-internet applications you can see right away when you make progress and that gave me the confidence and power to push back when I was frustrated.

Routes:

  • Login.jsx - Login form.
  • Register.jsx - Register form.
  • Feed.jsx - User logged in can post a new post there and see a list of all other users' posts.
  • FollowingUsers.jsx- User logged in can search for other users by prefix-search and can follow/unfollow other users who registered for the web.
  • DogsTips.jsx- This page includes five tips for dog owners.
  • DogsParks.jsx- This page lists dog park options in different areas of Israel.
  • Admin.jsx- This page contains relevant admin operations: remove user, view user activity, and enable/disable features.
  • MainLayout.jsx- Wrapping the web with the features we wanted to appear on all pages.
  • UserList.jsx- Fetching the users, displaying them to the logged in user and having the option to follow/unfollow other users.
  • Logout.jsx- Delete the token of the current logged in user, and redirect to the login page.
  • HamburgerMenu.jsx- Helping to navigate through the app by displaying all the pages a user can visit.
  • DogOfTheDay.jsx- A feature of a picture box that gets the picture's data from the users.json file.
  • DarkMode.jsx- A feature where a picture is randomly selected from all the photos uploaded by users during registration and is displayed on the Feed Page. The picture changes every 24 hours.
